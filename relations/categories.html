EVALUATION OF DPM AND 2nd STAGE CLASSIFIER ON DIFFERENT CATEGORIES

Description
  • DPM trained using train+val data from PascalVOC 2012 challenge
  • 2nd Stage Classifier trained on SIFT+HOG+RGBHist, with 3 level SPM
  • Training the 2nd-SC on a "clean" dataset with low-entropy negative examples
  • Combination of scores using the procedure described here
  • Validation using Pascal10X (Deva's dataset) except Dog (our data)

Comments
  • Please, note that validation was conducted on images with the target object only
  • Proper validation should mix all categories to measure detector's ability to select correct objects
  • Pascal10X dataset does not have a clean and high-quality ground truth as expected
  • Some examples: annotation on incorrect object, missing annotations, boxes too big, one bbox covering 2 cats, poorly localized objects, weird annotations.
